免费图床：Gitee

发表于2023-12-16|更新于2024-09-07|免费图床图床
  1. 创建 Gitee 账号
    登录 Gitee 官网 创建账号。

设置个性化域名，这个随意即可。

  1. 创建一个新仓库
    点击创建新仓库。

在设置中，选择是否添加 README 文件，配置分支模型。

进入管理设置，滚动到最下方，将仓库设置为开源，并保存。【一定要设置为开源】

获取 Gitee 令牌：在 Gitee 的【个人设置】->【安全设置】->【私人令牌】中生成令牌，选择最基本的权限并输入登录密码。

生成令牌后，将其复制下来以备后用。

  1. 下载安装 PicGo
    从 PicGo Release 页面 下载并安装 PicGo。

安装 Gitee 插件。

安装完成后，打开 PicGo，看到如下界面：

配置 PicGo 的 Gitee 插件参数：

owner：填写仓库 URL 中 http://gitee.com/ 和 /仓库名 之间的部分
repo：填写仓库地址后缀
token：填写复制的 Gitee 私人令牌
其他设置保持默认，点击确定。如果你只有一个仓库，可以将其设置为默认仓库。

看到这里基本上你就已经搭建好了，不过为了方便还是要继续设置一些东西

自定义快捷键设置为自己常用的，更新助手可以关闭，上传前是否重命名根据个人需求配置。

  1. 使用方法和效率小技巧
    使用截图工具进行截图。QQ 截图的快捷键是 Ctrl+Alt+A，电脑自带截图工具通常使用 PrtSc。
    使用设置的快捷键 Ctrl+Shift+P 将图片快速上传到图床。
    在需要使用图片的地方，直接使用 Ctrl+V 进行粘贴。
    到此为止，你已经搭建好了自己的图床。
