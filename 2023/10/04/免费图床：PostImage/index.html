免费图床：PostImage

发表于2023-10-04|更新于2024-01-12
|阅读量:

简介

Postimage是一款来自国外的图片上传/托管工具，分为免费版和付费版，支持网页/win端/浏览器插件上传图片。其免费版的限制如下：

  • 热链接图片缩放至1280px
  • 最大上传 24Mb (3MB) 的图片

环境搭建

  1. 搜索插件 postimage 安装

  2. 前往 官网 注册账号

  3. 在 这里 获取 API KEY，点击生成后复制下来

  4. 在设置中填写相关信息

    • API Key：刚刚你复制的那串字符

    • 图片分配：可以看成文件夹，比如上述配置 postimage 就会把图片存储在 image 文件夹下

  5. 点击确定，设置完成！
